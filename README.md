# OPPO-A15s-Fix-Tool
This DA(Download Agent) for fix oppo a15s and force boot to user space in EL0(User Space Exception Level) and also it uses modified Trust Zone(TZ) in TEE (Trust Execution Environment) as TOS(Trust Operating System) .
Also,It allows to temp root device and unlock bootloader without ereasing any partition and force critical unlock because it uses modified lk(little kernel).
Also,This DA unbricks dead boot devices and disables SLA(Serial Link Auth) , DAA(Download Agent Auth) and SBC(Secure Boot) in boot.
